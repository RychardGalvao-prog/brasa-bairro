# Reference Locks — Capital Food Ops v2

As referências não são decoração. Cada uma tem função clara.

## 1. Burek / Awwwards

Função:

- estrutura de app/kiosk;
- pedido online;
- navegação viva;
- produto protagonista;
- menu horizontal/incomum;
- primeira dobra com menos texto.

Não copiar:

- layout literal;
- assets;
- textos;
- proporções exatas.
