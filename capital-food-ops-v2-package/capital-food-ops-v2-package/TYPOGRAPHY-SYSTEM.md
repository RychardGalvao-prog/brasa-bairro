# Typography System

## Sistema aprovado

```txt
Logo: logo oficial Brasa & Bairro
Display: Fraunces
UI/Leitura: Manrope
Detalhe humano: Caveat
```

## Fraunces

Uso:

- headlines grandes;
- chamadas editoriais;
- preço de destaque;
- momentos cinematográficos.

Função:

- criar alma gastronômica/editorial;
- fugir de fonte SaaS;
- dar presença premium.

## Manrope

Uso:

- UI;
- botões;
- menus;
- cardápio;
- formulário;
- textos de apoio.

Função:

- legibilidade;
- modernidade;
- neutralidade quente sem virar tech demais.

## Caveat

Uso mínimo:

- pequenas anotações;
- setas;
- “feito na chapa”;
- “sai muito”;
- detalhes humanos.

Proibido:

- usar em blocos longos;
- usar em botões principais;
- usar como estética infantil.

## Preço

Preço deve ter peso visual. Ele deve parecer cardápio/cartaz, não label de SaaS.

## Hierarquia

```txt
Logo/selo
Headline display
Subheadline curta
CTA
Tags humanas
Preço
Detalhes operacionais
```
